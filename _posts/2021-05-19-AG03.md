# 1. 이항 계수

- 비효율적인 코드

```c
//nCk
int bin(int n, int k)
{
    if(k==0||n==k)
    {
        return 1;
	}
    else
        return bin(n-1, k-1)+bin(n-1, k);
}
```

```c
int bin(int n, int k)
{
    int i, j;
    int B[n][k];
    
    for(i =0; i<=n; i++)
        for(j=0; i<min(i, k); j++)
        {
            if(j==0||j==i)
            {
                B[i][j] = 1;
            }
            else
                B[i][j] = B[i-1][j-1] + B[i-1][j];
        }
    return B[n][k];
}
```
