복사 생성자
생성자의 Default는 얕은 복사이기 떄문에
직접 깊은 복사를 하는 복사생성자를 만들어주어야한다.
------------------------------------------*/

class CopyConstructor
{
private:
	int a;
	int* b;
	int* c;
public:
	CopyConstructor(const CopyConstructor &copy) {
		a = copy.a;
		b = new int;
		*b = *copy.b;
		c = new int;
		memcpy(c, copy.c, sizeof(int));
	}
};
